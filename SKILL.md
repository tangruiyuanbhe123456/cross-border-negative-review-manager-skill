# 跨境电商差评管理工具 (Negative Review Manager)

自动分析Amazon/Shopee/Temu的差评，生成AI优化回复方案，帮助卖家降低退货率、提升BSR排名。

## 功能

- 🔍 **差评自动分析**：输入差评内容，自动分析客户真实痛点
- ✍️ **智能回复生成**：根据差评类型生成合规的亚马逊回复模板
- 📊 **趋势分析**：分析近期差评关键词，发现产品问题
- 🛠 **Listing优化建议**：根据差评反馈，生成listing优化方案
- 📈 **竞品差评分析**：分析对手差评，找到自己的差异化卖点

## 使用方法

### 1. 分析单条差评

告诉Agent：
```
分析这条差评：[粘贴差评内容]
```

Agent会自动返回：
- 客户真实诉求分析
- 建议回复模板（中文+英文）
- 是否需要修改Listing
- 是否需要联系客户

### 2. 批量分析差评趋势

```
分析这些差评的共同问题：[粘贴多条差评]
```

返回：
- 高频问题TOP5
- 退货原因分析
- 紧急程度排序
- 改进优先级

### 3. 竞品差评分析

```
分析这个ASIN的差评：[ASIN码]
```

返回：
- 竞品产品痛点
- 我们可以优化的方向
- 市场机会点

## 适用平台

- Amazon（美国站/欧洲站/日本站）
- Shopee（东南亚站）
- Temu
- 独立站

## 最佳实践

1. **差评处理黄金24小时**：发现差评后24小时内处理，对恢复评分最有效
2. **不要直接索要好评**：Amazon政策严禁用物质回报换取好评
3. **差评是产品改进信号**：连续3条以上同样投诉，应考虑修改产品
4. **保持专业语气**：回复中不要出现情绪化语言
